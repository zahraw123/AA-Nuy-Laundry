# AA NUY Laundry - Admin Panel

Aplikasi web admin untuk manajemen laundry berbasis React + Vite.

## Cara Menjalankan

```bash
npm install
npm run dev
```

## Login
- Username: `admin`
- Password: `admin123`

## Fitur
- 📊 Dashboard dengan statistik dan quick action
- 📝 Form Pemesanan - tambah order baru dengan kalkulasi otomatis
- 💳 Form Pembayaran - proses pembayaran dengan alur step-by-step
- 🔄 Status Cucian - update dan pantau progress order
- 👥 Detail Pesanan - kelola semua detail pesanan
- 📈 Laporan - rekap pendapatan dan analisis

## Deploy ke Vercel
1. Push ke GitHub
2. Import repo di vercel.com
3. Deploy (otomatis terdeteksi sebagai Vite project)

## Tech Stack
- React 18
- React Router DOM v6
- Context API (state management)
- Tailwind CSS
- Vite
